Data Science Project

Project Description:
In a group project for my CMSC320 (Introduction to Data Science) course, I led my team in building an end-to-end data science pipeline analyzing 9,700+ films by integrating Letterboxd and TMDB data to identify and quantify which movie attributes most influence audience ratings. We implemented and evaluated multiple regression models covered in class, ultimately optimizing a Random Forest model to predict ratings and highlight influential features. 		Technical Challenges:
Addressed challenges such as interpreting the TMDB data schema and accurately merging Letterboxd and TMDB datasets using title and release date, resolving inconsistencies across sources. Ensured data quality by removing duplicates, handling missing values, and mitigating outliers to support reliable analysis.
Tools Used:
Python, Pandas, NumPy, SQLite, Matplotlib, Scikit-Learn, PyTorch 		Project Link:

MNIST Project

Project Description:
I built an end-to-end deep learning pipeline using PyTorch to train and evaluate a neural network for handwritten digit recognition on the MNIST dataset. The goal of the project was to gain hands-on experience designing neural networks, training models on real image data, and analyzing model performance through visualization and error analysis. The project included dataset exploration, model training with validation-based checkpointing, test evaluation, and interpretability through confusion matrices and misclassification analysis. 		Technical Challenges:
Addressed challenges in training and interpreting deep learning models by building an end-to-end PyTorch pipeline for handwritten digit classification on the MNIST dataset. Evaluated model behavior using learning curves, confusion matrices, and misclassification analysis to diagnose training dynamics, identify failure cases, and gain insight into neural network performance on real image data.
